# Brand Kit — 브랜드 가이드라인 입력

영상 제작 시 브랜드 일관성을 유지하기 위해 로고, 색상, 폰트, 톤 가이드라인을 수집하고 문서화합니다.

## 프로세스

### Step 1: 브랜드 가이드라인 유무 확인

> "기존 브랜드 가이드라인(CI 매뉴얼)이 있나요?\n1. 있다 — 파일이나 URL로 공유 가능\n2. 부분적으로 있다 — 로고와 색상 정도만\n3. 없다 — 이번에 새로 정하면 됨"

### Step 2: 브랜드 요소 수집

`rules/questioning-protocol.md`의 모호함 감지 규칙을 적용합니다.

**필수 질문 항목:**

| # | 항목 | 질문 |
|---|------|------|
| 1 | 로고 | "로고 파일이 있나요? (PNG/SVG 투명 배경 권장)\n- 있으면 파일 경로를 알려주세요\n- 없으면 텍스트 로고를 만들 수 있습니다" |
| 2 | 로고 배치 규칙 | "로고 배치에 대한 규칙이 있나요?\n1. 좌상단 고정\n2. 우하단 고정\n3. 인트로/아웃트로에만 표시\n4. 특별한 규칙 없음" |
| 3 | 주요 색상 | "브랜드의 주요 색상(Primary Color)은 무엇인가요?\n- HEX 코드가 있으면 알려주세요 (예: #FF6B00)\n- 없으면 어떤 색 계열인지 알려주세요" |
| 4 | 보조 색상 | "보조 색상(Secondary Color)이 있나요?\n- 있으면 HEX 코드 또는 색 계열\n- 없으면 주요 색상 기반으로 제안합니다" |
| 5 | 폰트 | "지정된 폰트가 있나요?\n1. 있다 — 폰트명을 알려주세요\n2. 없다 — 영상 스타일에 맞게 제안합니다" |
| 6 | 톤 오브 보이스 | "브랜드의 말투/톤은 어떤 느낌인가요?\n1. 전문적이고 공식적 (대기업, 공공기관)\n2. 친근하고 편안 (스타트업, B2C)\n3. 열정적이고 에너지 넘치는 (스포츠, 엔터)\n4. 신뢰감 있고 차분한 (금융, 의료, 교육)" |
| 7 | 금지 요소 | "브랜드에서 절대 사용하면 안 되는 것이 있나요?\n(특정 색상, 이미지 유형, 표현, 경쟁사 언급 등)" |

### Step 3: 로고 파일 확인 및 복사

로고 파일이 제공된 경우:
```bash
mkdir -p public/brand
cp [로고 경로] public/brand/
ls public/brand/
```

### Step 4: 브랜드 가이드라인 문서 작성

```bash
mkdir -p docs
```

**출력 파일**: `docs/00-brand-guidelines.md`

```markdown
# Brand Guidelines

## 로고
- 파일 경로: `public/brand/[파일명]`
- 배치 규칙: [규칙]
- 최소 크기: [크기]
- 여백 규칙: [규칙]
- 사용 금지 배경: [있으면 기재]

## 색상 팔레트
| 용도 | 색상명 | HEX | RGB | 사용 가이드 |
|------|--------|-----|-----|------------|
| Primary | [이름] | [#XXXXXX] | [R,G,B] | [주 사용처] |
| Secondary | [이름] | [#XXXXXX] | [R,G,B] | [보조 사용처] |
| Accent | [이름] | [#XXXXXX] | [R,G,B] | [강조 사용처] |
| Background | [이름] | [#XXXXXX] | [R,G,B] | [배경 사용처] |
| Text | [이름] | [#XXXXXX] | [R,G,B] | [텍스트 색상] |

## 타이포그래피
- 제목 폰트: [폰트명] — [크기 범위] — [굵기]
- 본문 폰트: [폰트명] — [크기 범위] — [굵기]
- 자막 폰트: [폰트명] — [크기 범위] — [굵기]
- 폰트 파일 경로: `public/brand/fonts/` (있을 경우)

## 톤 오브 보이스
- 말투: [설명]
- 존칭 여부: [설명]
- 키워드: [브랜드를 나타내는 형용사 3~5개]
- 금지 표현: [목록]

## 비주얼 스타일
- 이미지 톤: [설명]
- 선호하는 이미지 유형: [사진/일러스트/3D 등]
- 금지 이미지 유형: [목록]

## 금지 요소
- [항목 1]
- [항목 2]
- [항목 3]

## 브랜드 체크리스트
- [ ] 로고가 가이드라인에 맞게 배치되었는가
- [ ] 색상이 브랜드 팔레트 내에 있는가
- [ ] 폰트가 지정된 것을 사용하는가
- [ ] 톤이 브랜드 가이드에 맞는가
- [ ] 금지 요소가 포함되지 않았는가
```

### Step 5: 사용자 확인

> "브랜드 가이드라인을 정리했습니다. 확인해주세요. 이 가이드라인은 이후 모든 단계에서 참조됩니다."
