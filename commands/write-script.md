# Write Script — 스크립트 작성

이전 문서들을 기반으로 장면별 비주얼, 나레이션, 자막, 음악/효과음 지시를 포함하는 AV 포맷 스크립트를 작성합니다.

**이 단계는 승인 체크포인트입니다.** 사용자의 승인 없이 다음 단계로 진행할 수 없습니다.

## 사전 조건

다음 파일이 모두 존재해야 합니다:
- `docs/01-client-brief.md`
- `docs/02-creative-brief.md`
- `docs/03-treatment.md`

없으면 해당 커맨드를 먼저 실행하도록 안내합니다.

## 프로세스

### Step 1: 이전 문서 읽기

```bash
cat docs/01-client-brief.md
cat docs/02-creative-brief.md
cat docs/03-treatment.md
```

### Step 2: 스크립트 방향 질문

스크립트를 작성하기 전에 사용자에게 **질문**합니다. `rules/questioning-protocol.md`의 모호함 감지 규칙을 적용합니다.

**필수 질문 항목:**

| # | 항목 | 질문 |
|---|------|------|
| 1 | 나레이션 화법 | "나레이션의 화법은 어떤 것이 좋을까요?\n1. 설명체 — '~입니다, ~합니다' (공식적, 교육)\n2. 대화체 — '~해요, ~거든요' (친근, 캐주얼)\n3. 명령/지시체 — '~하세요, ~하십시오' (안전, 매뉴얼)\n4. 내러티브 — '~했다, ~이었다' (스토리텔링, 다큐)" |
| 2 | 장면 구성 | "전체 영상의 구조는 어떻게 가져가면 좋을까요?\n1. 순차적 — 단계별로 차근차근 (절차 설명, 튜토리얼)\n2. 문제→해결 — 문제 제시 후 해결 방법 제시\n3. 스토리 — 인물 중심 내러티브 전개\n4. 하이라이트 — 핵심 장면 먼저, 세부 내용 후" |
| 3 | 강조 포인트 | "영상에서 특히 강조해야 할 부분이 있나요? (반복 설명, 텍스트 강조, 슬로우모션 등)" |
| 4 | 장면 수/밀도 | "장면 전환이 빠른 것과 느린 것 중 어떤 것을 선호하시나요?\n1. 빠른 전환 — 5초 이하 짧은 컷 위주 (에너지, 임팩트)\n2. 보통 — 5~15초 단위 (안정적, 교육)\n3. 느린 전환 — 15초 이상 긴 컷 (몰입, 시네마틱)" |
| 5 | 자막/텍스트 | "화면에 자막이나 핵심 텍스트를 표시할 건가요?\n1. 나레이션 전체 자막 (접근성)\n2. 핵심 키워드만 텍스트 오버레이\n3. 자막 없이 영상만\n4. 둘 다 (나레이션 자막 + 핵심 키워드)" |

**조건부 질문:**
- 교육 영상인 경우: "각 단계마다 요약이나 체크포인트가 필요한가요?"
- 인물이 등장하는 경우: "인물의 대사가 있나요? 아니면 나레이션만으로 진행하나요?"
- 5분 이상 영상인 경우: "중간에 챕터 구분이 필요한가요?"

모든 방향이 확정된 후 스크립트를 작성합니다.

### Step 3: 스크립트 작성

AV(Audio-Visual) 포맷으로 장면별 스크립트를 작성합니다. **모든 씬에 대해 아래 항목을 반드시 포함**합니다.

**출력 파일**: `docs/04-script.md`

```markdown
# Script

## 기본 정보
- 총 러닝타임: [예상 시간]
- 총 씬 수: [개수]
- 화면 비율: [16:9 / 9:16 / 1:1]
- 나레이션 언어: [한국어/영어]

---

## Scene 1: [씬 제목]

| 항목 | 내용 |
|------|------|
| 시간 | 0:00 ~ 0:05 |
| 예상 러닝타임 | 5초 |

### Visual (화면)
[이 장면에서 무엇이 보이는지 상세 서술]

### Audio (오디오)
**나레이션:**
> "[나레이션 전문 — 정확한 대사를 기재]"

**화면 자막/텍스트:**
- [화면에 표시될 텍스트가 있으면 기재]

**음악:**
[이 장면의 음악 지시 — 시작, 유지, 페이드 등]

**효과음:**
[필요한 효과음과 타이밍]

---

## Scene 2: [씬 제목]

| 항목 | 내용 |
|------|------|
| 시간 | 0:05 ~ 0:15 |
| 예상 러닝타임 | 10초 |

### Visual (화면)
[화면 설명]

### Audio (오디오)
**나레이션:**
> "[나레이션 전문]"

**화면 자막/텍스트:**
- [화면 텍스트]

**음악:**
[음악 지시]

**효과음:**
[효과음 지시]

---

[이하 모든 씬 동일 형식으로 반복]

---

## 전체 나레이션 스크립트 (연결)

> [전체 나레이션을 이어서 작성 — TTS 생성 시 이 텍스트를 사용]

## 오디오 요약

| 항목 | 내용 |
|------|------|
| 나레이션 총 글자 수 | [개수] |
| 예상 나레이션 시간 | [분:초] |
| 배경음악 구간 | [전체 / 특정 구간] |
| 효과음 개수 | [개수] |
```

### Step 4: 승인 요청

스크립트를 사용자에게 제시하고 **반드시 승인을 받습니다**:

> "스크립트를 작성했습니다. 나레이션 대사와 장면 구성을 확인해주세요. 수정이 필요한 부분이 있으면 말씀해주세요."

**승인 시**: 다음 단계(`/storyboard`)로 진행
**수정 요청 시**: 피드백 반영 후 다시 승인 요청
